# Password Generator (Python)
A simple Python program that generates a secure random password based on user-defined length, including letters, digits, and special characters.
## Features
* Generates strong random passwords
* User can choose password length (16–64 characters)
* Includes:
  * Uppercase & lowercase letters
  * Numbers
  * Special characters (!@#$%^&*)
* Input validation for secure length range
## How It Works
* The user enters a password length
* The program ensures it is between **16 and 64**
* It combines letters, digits, and symbols
* A secure password is randomly generated and displayed
## 🛠️ Technologies Used
* Python
* `random` module
* `string` module
